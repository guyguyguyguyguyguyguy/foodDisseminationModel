Food dissemination model describing the propogation of food in an ant nest.

To run create python virtual environment with the package requirements:
	- python -m venv path/to/model/myenv
	- go to local model directory
	- source myenv/bin/activate
	- pip install -r requirements.txt

To run GUI:
	- go to model directory
	- compile GUI class:	javac src/main.java
	- run main clas:	java main
